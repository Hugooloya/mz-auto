# MZ Auto Detail

This is the official landing page for **MZ Auto Detail**, a premium automotive detailing service. The website showcases their services, project gallery, and contact information.

## Tech Stack

This project is built with a modern headless architecture:

*   **Frontend**: [Astro](https://astro.build/) - A high-performance web framework for content-driven websites.
*   **Backend (CMS)**: [Sanity](https://www.sanity.io/) - A unified content platform for structured content.

## Project Structure

*   `frontend/`: Contains the Astro application code (Pages, Components, Styles).
*   `backend/`: Contains the Sanity Studio configuration and schemas.

## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone <repo-url>
    cd mz-auto
    ```

2.  **Frontend Development:**
    ```bash
    cd frontend
    pnpm install
    pnpm run dev
    ```

3.  **Backend Development:**
    ```bash
    cd backend
    pnpm install
    pnpm run dev
    ```
